# 朋友圈

朋友圈功能提供广告拦截、内容增强和更多自定义能力。

## 功能列表

- [拦截朋友圈广告](remove-moments-ads.md) — 拦截朋友圈信息流中的广告
- [拦截朋友圈删除](anti-moments-delete.md) — 拦截他人删除朋友圈并添加删除标记
- [朋友圈查询增强](enhance-query.md) — 显示历史撤回及缓存过的朋友圈内容
- [朋友圈伪集赞](fake-moments-likes.md) — 自定义朋友圈点赞的用户列表
- [消息转圈](forward-messages-to-moments.md) — 将聊天中的消息转发到朋友圈
- [朋友圈底部详细信息](display-details.md) — 在朋友圈列表项底部显示详情信息
- [单击不关闭视频播放器](no-close-video-player-on-click.md) — 点击朋友圈视频时展开控制栏而非关闭播放器
